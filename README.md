# Retail Sales Dashboard (Advanced Excel)

## Overview

This project is an interactive Retail Sales Dashboard built using Microsoft Excel.
The dashboard analyzes sales performance across multiple business dimensions including:

- Sales
- Profit
- Orders
- Profit Margin
- Category
- Region
- Segment
- Sub Category
---

## Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- GETPIVOTDATA
- Slicers
---

## Dashboard Features

✔ Dynamic KPI Cards
✔ Monthly Sales Trend
✔ Sales by Category
✔ Sales by Region
✔ Sales by Segment
✔ Profit by Sub Category
✔ Interactive Slicers
